## 📌 Project Overview

SmartWay Academy ERP is a full-stack web application built to digitize the day-to-day operations of coaching institutes.

The application helps administrators manage students, fees, payments, receipts, WhatsApp notifications, and academic records through a centralized dashboard.

The project is designed as a production-ready ERP and is being continuously upgraded with AI-powered educational features.

# ⚙ Local Installation

Clone Repository

```bash
git clone https://github.com/sachin7con/smartwayacademy.git
```

Install dependencies

```bash
npm install
```

Configure Environment Variables

```env
PORT=

MONGO_URI=

JWT_SECRET=

WHATSAPP_TOKEN=
```

Run Project

```bash
npm run dev
```
